# uoftbirdcourses
A single page web app designed with Vue.js that suggests bird courses to take for students at all three UofT campuses.


## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run your tests
```
npm run test
```
